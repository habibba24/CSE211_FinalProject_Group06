# EventsX Website Project Requirements

## Core Technologies Used:

### Frontend Technologies:
- **HTML5** - For semantic markup and page structure
- **CSS3** - For styling and responsive design
- **JavaScript (ES6+)** - For dynamic interactivity and functionality

### Browser Requirements:
- Chrome (recommended for full functionality)
- Firefox
- Safari
- Edge

### Development Tools:
- Visual Studio Code (recommended editor)
- Chrome DevTools (for debugging)
- Lighthouse (for performance analysis)

### Key Features Implemented:
- Event listing and booking system
- User registration and login
- Budget calculator
- Responsive design
- Form validation
- Dynamic search functionality

### No External Dependencies:
This project uses pure HTML, CSS, and JavaScript without any external frameworks or libraries.
