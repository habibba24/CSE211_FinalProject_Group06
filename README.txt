EventsX - Event Planner Website

This website is about booking and planning events in Egypt. Users can:
- Browse and book tickets for concerts (like Amr Diab, Elissa, Cairokee)
- Watch sports events (like Al Ahly vs Zamalek matches)
- Find theater shows and live performances
- Plan their own events
- Calculate budgets for events
- Register and login to manage bookings
